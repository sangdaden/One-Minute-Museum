
# UI Guidelines — One-Minute Museum

## Design Concept

Use **Concept 1: Heritage Warm** as the main visual direction.

One-Minute Museum should feel like a small modern Vietnamese museum: warm, cultural, friendly, trustworthy, and easy to share. The interface should combine Vietnamese heritage materials such as giấy dó, sơn son, vàng đồng, Đông Sơn patterns, lotus motifs, and traditional decorative lines with a clean modern app layout.

## Color Palette

| Role | Name | Hex | Usage |
|---|---|---:|---|
| Background | Kem giấy | `#F7F1E3` | Main page background |
| Primary | Đỏ son | `#A8322A` | Primary CTA, logo, active nav, important highlights |
| Secondary | Vàng đồng | `#C89B3C` | Fun fact badge, small icons, dividers, decorative accents |
| Text | Nâu than | `#2F2621` | Main text color |
| Accent | Xanh ngọc | `#2F7C74` | Secondary highlights, avatar, success/info states |
| Card | Trắng ngà | `#FFF9EF` | Cards, panels, input areas |
| Border | Be nâu nhạt | `#E3D2B8` | Card borders, separators, subtle outlines |

## Visual Style

- Use warm cream background, not pure white.
- Use rounded cards with soft shadows.
- Primary buttons should use `#A8322A` with white/cream text.
- Fun fact labels should use `#C89B3C`.
- Use `#2F2621` instead of pure black for text.
- Use `#2F7C74` sparingly for modern contrast.
- Add subtle Vietnamese heritage motifs, but avoid making the UI look too decorative or festival-like.
- Prefer clean spacing, editorial layout, and museum-card feeling.

## Components Direction

### Primary Button

- Background: `#A8322A`
- Text: `#FFF9EF`
- Border radius: large / pill style
- Shadow: subtle warm shadow

### Cards

- Background: `#FFF9EF`
- Border: `1px solid #E3D2B8`
- Border radius: 16px–24px
- Shadow: soft, warm, not harsh

### Fun Fact Badge

- Background: `#C89B3C`
- Text: `#FFF9EF`
- Optional icon: lightbulb, sparkle, drum, lotus

### Page Background

- Background: `#F7F1E3`
- Optional subtle texture using gradient or paper-like pattern
- Avoid plain white backgrounds

## Typography

- Heading should feel editorial and cultural.
- Suggested heading font: `Playfair Display`, `Lora`, or a serif font.
- Body text should be clean and readable.
- Suggested body font: `Inter`, `Geist`, or system sans-serif.

## Product Feeling

The product should communicate:

- Nhanh gọn
- Văn hóa
- Truyền cảm hứng
- Dễ chia sẻ
- Có cảm giác như một bảo tàng mini hiện đại
