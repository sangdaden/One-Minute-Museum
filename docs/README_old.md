# Bảo Tàng 1 Phút — One-Minute Museum

**Bảo Tàng 1 Phút** là một web app biến những vật bình thường quanh người dùng thành một “triển lãm mini” có thể đọc trong 1 phút.

Ví dụ:
- Dép tổ ong
- Ghế nhựa đỏ
- Ly cà phê sữa đá
- Remote TV bọc nilon
- Áo mưa
- Bàn phím cơ
- Dây sạc điện thoại

## Mục tiêu MVP

Người dùng nhập tên một vật, chọn góc nhìn, sau đó AI tạo ra một mini exhibition gồm:
- Tiêu đề
- Hook mở đầu
- Mô tả vật thể
- Câu chuyện / nguồn gốc / bối cảnh
- 3 fun facts
- Insight thiết kế / văn hóa
- Vì sao vật này đáng chú ý
- Câu hỏi suy ngẫm
- Quote dùng để tạo share card

## Hướng triển khai ưu tiên

MVP nên tập trung vào niche:

> Vietnamese Everyday Museum — Bảo tàng những vật bình thường rất Việt Nam.

## Bộ file trong thư mục này

- `spec.md`: đặc tả sản phẩm tổng thể.
- `mvp_scope.md`: phạm vi MVP và non-goals.
- `user_stories.md`: danh sách user stories theo sprint.
- `feature_specs.md`: đặc tả chi tiết từng tính năng.
- `prompt_spec.md`: prompt JSON output cho AI.
- `api_spec.md`: API contract đề xuất.
- `data_model.md`: database schema đề xuất.
- `ui_spec.md`: đặc tả giao diện.
- `build_in_public_plan.md`: kế hoạch đăng bài 7 ngày.
- `coding_agent_instructions.md`: hướng dẫn cho AI coding agent/dev team.
