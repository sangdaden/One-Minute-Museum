# Design — UI internationalization (vi↔en)

**Date:** 2026-06-11
**Status:** Approved (pending spec review)
**Library:** next-intl, **without i18n routing** (cookie-based locale, no URL change).

## 1. Quyết định đã chốt

- **next-intl**, chế độ không routing: locale theo **cookie `omm-locale`** (mặc định `vi`,
  nhận `vi|en`), đọc per-request. Giữ nguyên mọi route hiện có.
- Toggle ngôn ngữ trong **AccountMenu** (cạnh Sáng/Tối): set cookie + `router.refresh()`.
- **Chỉ dịch giao diện.** Nội dung AI (title/hook/facts…) vẫn theo ngôn ngữ tạo (vi).
- Giá trị canonical (mode/voice/theme lưu DB) **không đổi**; chỉ dịch **nhãn hiển thị + mô tả**.
- Nhãn voice EN: Researcher · Storyteller · Street vendor · Poet.
- **Chia 3 phase, ship từng phase.**

## 2. Setup (Phase 1)

- Cài `next-intl`.
- `i18n/request.ts` — `getRequestConfig`: đọc cookie `omm-locale` → `locale` (vi|en) +
  `messages = (await import('../messages/${locale}.json')).default`.
- `next.config.ts` — bọc `createNextIntlPlugin()` (mặc định trỏ `./i18n/request.ts`),
  giữ `allowedDevOrigins`.
- `app/layout.tsx` (async): `getLocale()` + `getMessages()` → `<html lang={locale}>` +
  bọc `children` trong `<NextIntlClientProvider messages>`. Giữ script no-FOUC theme.

## 3. Đổi ngôn ngữ

- AccountMenu thêm hàng **"Ngôn ngữ"**: `useLocale()` để biết hiện tại; bấm →
  `document.cookie = "omm-locale=<vi|en>; path=/; max-age=31536000"` + `router.refresh()`.

## 4. Messages

- `messages/vi.json` + `messages/en.json`, namespace: `Common`, `Menu`, `Masthead`,
  `Create`, `Modes`, `Voices`, `Themes`, `Card`, `Feed`, `Post`, `Profile`, `Gallery`, `Errors`.
- Server component: `const t = await getTranslations('Ns')`. Client: `const t = useTranslations('Ns')`.
- Mode/voice: map từ giá trị canonical → key label/description trong `Modes`/`Voices`.

## 5. Phasing (mỗi phase: tsc + build + verify + merge)

1. **Setup + chrome dùng chung**: next-intl + cookie + toggle; dịch `AccountMenu`,
   masthead (Khám phá/Trang chủ/Bộ sưu tập…), nút chung (Copy, Đăng, Tạo lại, Tải ảnh,
   Sửa nội dung, Lưu/Hủy).
2. **/create**: hero, nhãn section, mô tả `MODE_META`/`VOICE_META`, placeholder input,
   `EditExhibitionForm`, `LoadingExhibition`, `ErrorState`, PublishButton, ThemePicker,
   ghi chú note, SuggestedObjects header.
3. **Feed/Chi tiết/Profile/Gallery/Me + components**: nhãn `ExhibitionCard`/`ThemedCard`,
   `FeedPost`, `CommentForm`/`CommentList`, `ReactionBar` (REACTIONS labels), `ShareCard`
   nhãn, empty states.

Phần chưa dịch ở phase sau tạm hiển thị tiếng Việt (không vỡ) khi locale=en.

## 6. Verify

- Mỗi phase: `tsc` + `build`; đặt cookie `omm-locale=en` → phần đã dịch ra tiếng Anh,
  phần chưa dịch giữ tiếng Việt. Đổi qua lại bằng toggle trong menu.
- Không có Playwright — nhìn mắt qua trình duyệt.

## 7. Ngoài phạm vi

- Sinh **nội dung AI theo locale** (EN UI → EN exhibition) — cần chỉnh prompt; để sau.
- Locale trong URL / SEO theo locale (đã chọn cookie). Thêm ngôn ngữ thứ 3.
- Định dạng ngày/số theo locale (hiện `formatDate` cố định) — có thể thêm sau.
